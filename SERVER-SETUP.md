# Server Setup — Salon Booking Platform

## Stack Overview

| Layer     | Technology          | Version  |
|-----------|---------------------|----------|
| Backend   | Laravel             | 12.x     |
| Frontend  | Next.js             | 15.x     |
| Language  | PHP                 | ^8.2     |
| Runtime   | Node.js             | 20.x LTS |
| Database  | PostgreSQL          | 13+      |
| Mail      | Mailpit (local dev) | —        |

---

## 1. Prerequisites

### Local (Laragon)
- [Laragon](https://laragon.org/) with PHP 8.3
- Node.js 20.x LTS — via [nvm-windows](https://github.com/coreybutler/nvm-windows)
- PostgreSQL 13+ (bundled with Laragon or standalone)
- Composer 2.x
- Mailpit (bundled with Laragon) — UI at `http://localhost:8025`

### Production (Ubuntu/Debian)
```bash
# PHP 8.3
sudo apt install php8.3 php8.3-cli php8.3-fpm php8.3-pgsql php8.3-mbstring \
  php8.3-xml php8.3-curl php8.3-zip php8.3-bcmath php8.3-tokenizer

# Composer
curl -sS https://getcomposer.org/installer | php
sudo mv composer.phar /usr/local/bin/composer

# Node 20
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt install -y nodejs

# PostgreSQL
sudo apt install postgresql postgresql-contrib

# Nginx + PM2 (frontend)
sudo apt install nginx
npm install -g pm2
```

## 2. Backend Setup (Laravel)

```bash
cd salon-saas-backend
composer install --no-dev --optimize-autoloader
cp .env.example .env
php artisan key:generate
php artisan migrate --force
php artisan storage:link
```

### Environment Variables (`.env`)

| Variable              | Description                        | Example                        |
|-----------------------|------------------------------------|--------------------------------|
| `APP_KEY`             | Generated by `artisan key:generate`| `base64:...`                   |
| `APP_URL`             | Backend base URL                   | `http://localhost:8000`        |
| `DB_CONNECTION`       | Database driver                    | `pgsql`                        |
| `DB_HOST`             | DB host                            | `127.0.0.1`                    |
| `DB_PORT`             | DB port                            | `5432`                         |
| `DB_DATABASE`         | Database name                      | `salon_saas`                   |
| `DB_USERNAME`         | DB user                            | `postgres`                     |
| `DB_PASSWORD`         | DB password                        | `<password>`                   |
| `JWT_SECRET`          | JWT signing key                    | `<generate with artisan>`      |
| `MAIL_MAILER`         | Mail driver                        | `smtp` (local) / `ses` (prod)  |
| `MAIL_HOST`           | SMTP host                          | `127.0.0.1`                    |
| `MAIL_PORT`           | SMTP port                          | `1025` (Mailpit) / `587` (prod)|
| `MAIL_FROM_ADDRESS`   | Sender email                       | `noreply@yourdomain.com`       |
| `QUEUE_CONNECTION`    | Queue driver                       | `database`                     |
| `SESSION_DRIVER`      | Session driver                     | `database`                     |

Generate JWT secret:
```bash
php artisan jwt:secret
```

### Production: Nginx config for Laravel
```nginx
server {
    listen 80;
    server_name api.yourdomain.com;
    root /var/www/salon-saas-backend/public;

    index index.php;

    location / {
        try_files $uri $uri/ /index.php?$query_string;
    }

    location ~ \.php$ {
        fastcgi_pass unix:/var/run/php/php8.3-fpm.sock;
        fastcgi_param SCRIPT_FILENAME $realpath_root$fastcgi_script_name;
        include fastcgi_params;
    }
}
```

## 3. Frontend Setup (Next.js)

```bash
cd salon-saas-frontend
npm install
cp .env.local.example .env.local
# Edit .env.local — set NEXT_PUBLIC_API_URL
npm run build
```

### Environment Variables (`.env.local`)

| Variable               | Description              | Example                        |
|------------------------|--------------------------|--------------------------------|
| `NEXT_PUBLIC_API_URL`  | Backend API base URL     | `http://localhost:8000`        |

### Production: Run with PM2
```bash
npm run build
pm2 start npm --name "salon-frontend" -- start
pm2 save
pm2 startup
```

### Production: Nginx reverse proxy for Next.js
```nginx
server {
    listen 80;
    server_name yourdomain.com;

    location / {
        proxy_pass http://localhost:3000;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_cache_bypass $http_upgrade;
    }
}
```

## 4. Queue Worker (Production)

```bash
# Run as a systemd service or via supervisor
php artisan queue:work --sleep=3 --tries=3 --max-time=3600
```

Supervisor config (`/etc/supervisor/conf.d/salon-queue.conf`):
```ini
[program:salon-queue]
command=php /var/www/salon-saas-backend/artisan queue:work --sleep=3 --tries=3
autostart=true
autorestart=true
user=www-data
redirect_stderr=true
stdout_logfile=/var/log/salon-queue.log
```

## 5. Local Dev — Quick Start

```bash
# Terminal 1 — Backend
cd salon-saas-backend
composer run dev        # starts Laravel + queue + Vite + logs

# Terminal 2 — Frontend
cd salon-saas-frontend
npm run dev             # starts Next.js at http://localhost:3000
```

Backend runs at `http://localhost:8000`, frontend at `http://localhost:3000`.
